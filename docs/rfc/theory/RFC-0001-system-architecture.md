# RFC-0001 - System Architecture

Status: Draft
Section: Theory RFC
Version: v0.1

## Purpose

Defines the overall VET-C system architecture from perception to action.

## Scope

Includes high-level module roles, information flow, recurrent world modeling, regime inference, terrain-field generation, and policy output.

## Current MVP Status

This is currently a theory/specification placeholder. The Section 2 MVP implements a runnable subset through `VETCSystem`.

## Implementation Notes

The current architecture is reflected by `vetc/runtime/vetc_system.py` and the module packages under `vetc/`.

## Limitations

The full architecture from the theory roadmap is not implemented; memory, dreaming, density flow, and deployment systems are future work.
