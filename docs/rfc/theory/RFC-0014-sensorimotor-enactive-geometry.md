# RFC-0014 - Sensorimotor Enactive Geometry

Status: Future Work
Section: Theory RFC
Version: v0.1

## Purpose

Defines the planned sensorimotor and enactive geometry layer for VET-C.

## Scope

Includes relationships between perception, action, embodied state, and world-model geometry.

## Current MVP Status

This is currently a theory/specification placeholder. The MVP validates a perception-to-action tensor path only.

## Implementation Notes

Current related modules include `vetc/observation/`, `vetc/encoder/`, `vetc/rssm/`, and `vetc/policy/`.

## Limitations

No explicit enactive geometry module or objective exists yet.

## Future Work

Define formal contracts and integrate them with training and evaluation.
